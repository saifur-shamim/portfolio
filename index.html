My Publications

Streamlining Global Visa Processing and Document Verification: A Self-Sovereign Identity System for Secure and Efficient Visa Processing Using Hyperledger Indy.

2nd International Conference on Information and Communication Technology (ICICT) October 21-22, Dhaka, Bangladesh | 2024

This research introduces an SSI-based system using Hyperledger Indy to streamline visa processing and immigration verification. It employs Decentralized Identifiers (DIDs) and Verifiable Credentials (VCs) for secure, privacy-compliant communication among passport offices, visa centers, and immigration authorities. With Hyperledger Cacti enabling cross-border data exchange, the system reduces processing time, enhances security, and improves collaboration between countries.

Leveraging Self-Sovereign Identity (SSI) with Hyperledger Indy: A Decentralized Identity Ecosystem for Secure Document Management in Bangladesh.

27th International Conference on Computer and Information Technology, December 20-22 | 2024

This study presents a Self-Sovereign Identity (SSI) framework built on Hyperledger Indy to modernize document management in Bangladesh. It addresses inefficiencies and security gaps in obtaining vital records such as NID cards and birth certificates. By equipping citizens with Decentralized Identifiers (DIDs) and Verifiable Credentials (VCs), the system ensures secure, tamper-resistant document issuance and verification. The model supports selective data sharing to uphold privacy while integrating with governmental systems to simplify access and reduce administrative burdens. This innovative approach aims to create a trusted, citizen-centric identity ecosystem in Bangladesh.
